About

我叫劉奉一，英文名是Leo Liu，來自台北，現居台中，畢業於中興大學歷史研究所所。在求學期間，我感受到歷史系畢業後出路較其他科系狹窄，認為應該嘗試學習其他技能，做跨領域的結合，因而在退役後報名了職訓局的網 路行銷課。

在修課期間，短暫接觸到一些前端的基礎技能，發現自己對程式很有興趣，喜歡畫面操作，視覺的掌控，喜歡刻出網頁的每一個過程。結訓後，我決定繼續加強程式能力，參與了六角學院線上課程的培訓，以前端工程師為目標，不斷學習。

未來我也會持續精進前端的各種技能，並慢慢學習後端、及設計方面的一些基礎，讓自己能夠真正成為一個串聯設計到後端的前端工程師。
